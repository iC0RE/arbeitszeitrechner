Version 2.0.0

Highlight

  • Passend zur Adventszeit wurde erstmalig ein x-mas-Theme eingeführt.
  • Das Theme kann im oberen Abschnitt gewechselt werden.

Neu

  • Der Arbeitszeitrechner heißt ab sofort Dienstzeitrechner.
  • Das Wort Arbeit wurde überall durch das treffendere Wort Dienst ersetzt.
  • Das Design des Akkordions wurde dem Seitendesign angepasst.
  • Zur Vorauswahl wurde ein weiterer Standard-Wert (7:00) hinzugefügt.
  • Der Footer wurde am unteren Rand der Seite fixiert.
  • Die Labels haben neue, passendere Bezeichnnungen erhalten.

Version 1.2.0

Neu

  • Zur Vorauswahl der Regelarbeitszeit wurde eine Button-Group oberhalb der Eingabefelder hinzugefügt (Als Standard Regelarbeitszeit ist weiterhin 7:48h vorausgewählt).
  • Das Standard-Favicon wurde gegen ein neues Favicon ausgetauscht.
  • Zusätzlich zur Differenz wurde die prozentuale Anzeige der Arbeitszeit im Vergleich zur Regelarbeitszeit hinzugefügt.
  • Neu hinzugefügt wurde ebenso ein positives Vorzeichen bei positiver Differenz.
  • Das Akkordion wird nun standardmäßig ungeöffnet dargestellt.

Behoben

  • Der Standard-Wert für Pause wurde von "--:--" auf "00:00" geändert.
  • Ein Fehler, der dazu führte, dass bei identischer Arbeits- und Regelarbeitszeit eine negative Differenz angezeigt wird, wurde behoben.

Version 1.1.0

Neu

  • Das Jumbotron zur Ergebnisanzeige wird nun dauerhaft angezeigt.
  • Nach Eingabe des Beginns der Arbeitszeit wird das Feld "Ende" automatisch mit dem Soll-Ende gefüllt.
  • Javascript nutzt nun die js-Bibliothek jQuery.
  • Das Panel "Hinweise zum Arbeitszeitrechner" wurde entfernt.
  • Einführung des "Akkordions" zur Anzeige der Changelogs mit Verlinkungen zu Vorgängerversionen.
  • Anzeige von geplanten Features für die zukünftigen Versionen.
  • Button zur Einreichung weiterer Vorschläge (im Akkordion unter "Zukünftige Versionen").

Behoben

  • Berechnung von Arbeitszeiten über 9:00h funktioniert nun ohne falsche Berechnung.
  • Die Pausenzeit kann nun geändert werden, ohne das der automatisch der Standard-Wert eingetragen wird.
  • Änderungen der Soll-Zeit führen nicht mehr zu falschen Ergebnissen.

Version 1.0.0

Neu

  • Berechnung der reinen Arbeitszeit, abzüglich der Pause
  • Anzeige der Differenz zur Soll-Arbeitszeit
  • Vollautomatische Vorauswahl der Standard-Pausenzeit, sobald Beginn und Ende festgelegt sind
  • Festlegung einer anderen Soll-Arbeitszeit (Standard: 7:48h)
  • Autofocus auf Arbeitszeitbeginn bei jedem Laden der Seite
  • Anzeige von grundlegenden Hinweisen zum Arbeitszeitrechner (Hinweise können nur durch Neuladen der Seite ausgeblendet werden)
  • Verwendung des von Twitter Bootstrap zur Verfügung gestellten GRID-Systems zur Platzierung der Eingabemasken.
