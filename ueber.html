Wann kann ich Feierabend machen?

Der nächste Schritt

Plattformunabhängigkeit.

Die Vorteile einer Web-Anwendung sind die Plattformunabhängigkeit. Es wird kein bestimmtes Betriebssystem und keine properitäre Software benötigt. Außerdem ist Sie immer und überall verfügbar (eine Internetverbindung wird vorausgesetzt).

Version 1.0

Die erste publizierte Web-App.

Sie bestand nur aus der Transformation des Excel-Arbeitszeitrechners in eine Web-Anwendung und war ausschließlich auf die Berechnung der Arbeitszeit für eine 39-Stunden-Woche ausgelegt.

Bootstrap

Das weltweit beliebteste Framework.

Seit Beginn der Entwicklung wird für den Arbeitszeitrechner das Framework Bootstrap verwendet. Zunächst in Version 3.3.7. Seit dem Design-Update in Version 4.1 wird Bootstrap 4.0.0 eingesetzt.

Weiterentwicklung

Meilensteine des Arbeitszeitrechners.

Mit der Zeit wurde der Funktionsumfang des Arbeitszeitrechners ausgebaut und um (mehr oder weniger) wichtige Features erweitert.

In diesem Abschnitt werden einzelne Entwicklungsschritte etwas detaillierter beschrieben.

JavaScript

Implementierung der JS-Bibliothek jQuery.

In Verion 1.1 wurde die automatische Berechnung der Endzeit (an Hand der Soll-Zeit) eingeführt. Der AZR verwendet seitdem die Javascript-Bibliothek jQuery, was die Entwicklung sehr vereinfachte. Das Jumbotron zur Anzeige der Ergebnisse wird nun dauerhaft angezeigt. Für das ChangeLog wird ein Akkordion implementiert.

Das Favicon

Endlich kein Standard mehr.

In der folgenden Version (1.2) wurde das Standard-Favicon gegen ein neues, simples Clock-Icon ersetzt. Weiterhin wurden die Buttons zur Vorauswahl der Arbeitszeit, sowie die prozentuale Anzeige der Arbeitszeit, hinzugefügt. Zuletzt wurde das Favicon in Verison 4.2 erneuert.

x-mas Theme

Das neue (Alternativ)-Design.

Das xmas-Theme wurde mit Version 2.0 zu Beginn der Adventszeit im Jahr 2016 veröffentlicht. Außerdem wurde die Bezeichnung Dienstzeitrechner an Stelle von Arbeitszeitrechner eingeführt, ein neues Design für das ChangeLog-Akkordion sowie ein weiteres Vorauswahlfeld (7:00) hinzugefügt.

Automatisierung

Live-Update für Berechnungen.

Seit der 3. Version werden die berechneten Werte vollautomatisch bei jeder Änderung aktualisiert. Dadurch wurde der Komfort erhöht und die Effizienz des Rechners verbessert.

Große Neuerung

Der Countdown läuft.

Mit der 4. Edition des Dienstzeitrechners wurde auf Wunsch eines Ntzers ein Countown implementiert, der die verbleibende Arbeitszeit herunterzählt. Außerdem wurden Vorauswahl-Buttons für die Pausenzeit eingeführt.
